# MediBot (minimal)

Small Next.js app with a local JSON DB for video calls (Jitsi), prescriptions, and notifications. Minimal, scalable layout and small main files.

Quick start:

- cd c:\Users\Ashwath\Webs\medibot
- npm install
- npm run dev

Features:
- Patient login by phone
- Doctor login by phone + name + NMC
- Create/Join calls (Jitsi room)
- Doctor can select multiple patients and notify them
- Doctor can create prescriptions per patient
- Local JSON DB in `data/db.json`
